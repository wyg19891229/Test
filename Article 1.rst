1. Assuming you have Python already, `install Sphinx`_:

::

    pip install sphinx
    

2. Create a directory inside your project to hold your docs:

::

    cd /path/to/project
    mkdir docs

3. Run ``sphinx-quickstart`` in there:

::

    cd docs
    sphinx-quickstart
